# 🌍 Global Land Temperature Analysis

## 📌 Overview

Analysis of global land temperature trends

## 💾 Data Source

Kaggle:
https://www.kaggle.com/datasets/vijayvvenkitesh/global-land-temperatures-by-country

## 📊 Methods

- Data cleaning and preprocessing
- Rolling mean (MA_10, MA_50)
- Time series visualization with seaborn/matplotlib

## 📈 Key Insights

- Long-term warming trend observed
- Seasonal variability smoothed using moving averages

## 🛠️ Tools

- Python
- Pandas
- Seaborn
- Matplotlib
